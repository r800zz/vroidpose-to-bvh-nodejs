# vroidpose-to-bvh-nodejs
Convert a .vroidpose file(VRoid Studio) to BVH. node.js

# Requirements
Node.js

# Usage
```
node vroidpose_to_bvh.js input.vroidpose out.bvh
```
# Extended example
R800ZZ WebXR VRM Viewer
https://vr180g.com/viewer/viewer.php?l=en

"R800ZZ WebXR VRM Viewer" can convert .vroidpose files into VRMA (.vrma).
